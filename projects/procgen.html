Procedural Level Generation Tool

Overview - Solo Project

This Unity-based tool was built to streamline procedural level design. It combines multiple generation algorithms into a flexible hybrid system that empowers designers to quickly prototype or produce full-featured levels with consistency and variety.

With designer-defined layout rules and live iteration support, the tool gives control back to the creator while maintaining the benefits of randomness. Whether crafting dungeons, arenas, or exploration zones, this system balances unpredictability with structure.

Role: Systems Programmer / Tool Designer

Tech Stack: Unity / C# / Editor Tooling / WFC / Cellular Automata / Scholarly Research

My Contributions

  • Designed a hybrid procedural generation algorithm by combining BSP and cellular automata
  • Built a flexible Unity Editor tool for designers to tweak generation parameters live
  • Implemented support for rule-based object placement and tile constraints
  • Optimized generation logic for performance in both 2D and 3D applications
  • Documented the tool and trained teammates on effective usage
